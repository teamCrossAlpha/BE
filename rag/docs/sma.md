주의: 본 문서는 기술적 지표의 의미를 설명하기 위한 참고 자료이며, 특정 종목의 매수·매도·보유를 권유하지 않는다. 기술적 지표는 시장 상황과 함께 종합적으로 해석되어야 하며 단독 판단 기준으로 사용될 수 없다.

# SMA (Simple Moving Average, 단순이동평균)

## 1. 개요 및 정의

SMA(단순이동평균)는 특정 기간 동안의 자산 종가(closing price)를 산술 평균한 값이다.
가격의 단기 변동성을 평활화(smoothing)하여 추세 방향을 시각적으로 파악하는 데 활용된다.
새로운 봉이 생길 때마다 가장 오래된 데이터를 제거하고 최신 데이터를 추가해 평균이 "이동"하기 때문에 이동평균이라 부른다.
모든 데이터 포인트에 **동일한 가중치**를 부여하는 것이 특징이다.

---

## 2. 계산 방법

### 공식

```
SMA = (A1 + A2 + ... + An) / n

여기서:
  An = n번째 기간의 자산 가격(일반적으로 종가)
  n  = 계산에 포함되는 총 기간 수
```

### 계산 예시 — 5일 SMA

종가: 10, 11, 12, 11, 14
SMA = (10 + 11 + 12 + 11 + 14) / 5 = **11.6**

### 계산 예시 — 15일 SMA

- 1주차(5일): 20, 22, 24, 25, 23
- 2주차(5일): 26, 28, 26, 29, 27
- 3주차(5일): 28, 30, 27, 29, 28
- 합계: 392 / 15 = **26.13**

---

## 3. 추세 판단 및 해석

### 방향성으로 추세 파악

- **SMA가 우상향**: 가격 상승추세를 나타냄
- **SMA가 우하향**: 가격 하락추세를 나타냄
- SMA 선의 기울기가 클수록 추세의 강도가 강하다고 해석

### 단기/장기 SMA 비교

| 상황 | 의미 |
|------|------|
| 단기 SMA > 장기 SMA | 상승추세 환경 (단기 가격이 장기 평균보다 높음) |
| 장기 SMA > 단기 SMA | 하락추세 환경 (단기 가격이 장기 평균보다 낮음) |

### 지지선/저항선으로서의 SMA

- **상승추세에서**: 가격이 SMA에 닿으면 반등하는 경향이 있어 **동적 지지선** 역할
- **하락추세에서**: 가격이 SMA까지 반등했다가 다시 내려오는 경향이 있어 **동적 저항선** 역할
- 특히 50일, 200일 SMA는 기관투자자들이 주시하는 주요 지지/저항 수준

---

## 4. 주요 기간별 SMA

| 기간 | 분류 | 주요 활용 |
|------|------|----------|
| 10일 | 초단기 | 최근 가격 변화에 빠르게 반응; 단기 트레이더용 |
| 20일 | 단기 | 스윙 트레이더가 주로 활용; 약 1개월 거래일 반영 |
| 50일 | 중기 | 가장 널리 사용; 골든크로스/데드크로스의 핵심; 약 분기(3개월) 반영 |
| 100일 | 장기 | 중장기 추세 판단 |
| 200일 | 장기 | 가장 중요한 장기 SMA; 시장 전체 건강도 기준; 기관투자자 기준선 |

---

## 5. 시장 국면별 SMA 해석

스냅샷에서 현재 가격과 SMA 값들을 받았을 때, 그 배열 관계로 시장 국면을 진단한다.

### 가격과 SMA의 배열로 국면 진단

스냅샷에서 price, SMA20, SMA50, SMA200을 받으면 아래 기준으로 국면을 판단한다.

| 배열 상태 | 국면 | 해석 |
|---------|------|------|
| price > SMA20 > SMA50 > SMA200 | 강한 상승추세 | 모든 기간 지표가 정렬됨. 추세 추종 전략 유효 |
| price > SMA50 > SMA200 (단기 조정 중) | 상승추세 내 조정 | 단기 pullback. SMA50 지지 확인 후 매수 기회 |
| SMA50 > price > SMA200 | 상승추세 약화 | 단기/중기 SMA 아래로 내려옴. 추세 전환 가능성 모니터링 |
| SMA200 > SMA50 > price | 강한 하락추세 | 모든 기간 지표가 역배열. 추세 역행 매수 위험 |
| price > SMA200 (단, SMA50 < SMA200) | 하락추세 기술적 반등 | 데드크로스 이후 일시 반등. 추세 전환 확인 전 주의 |
| SMA20 ≈ SMA50 ≈ SMA200 (수렴) | 횡보장 | 방향성 부재. 브레이크아웃 방향 대기 필요 |

### 가격과 SMA의 거리로 과열/침체 판단

현재 가격이 SMA에서 얼마나 떨어져 있는지도 중요한 신호다.

- **가격이 SMA200보다 크게 위에 있을 때**: 과매수 영역. 평균으로 회귀(Mean Reversion) 가능성 증가. 단기 조정 위험.
- **가격이 SMA200보다 크게 아래에 있을 때**: 과매도 영역. 기술적 반등 가능성. 단, 하락추세 지속 중일 수 있음.
- **가격이 SMA 근처에 있을 때**: 지지/저항 검증 구간. 이 수준에서의 가격 행동(반등 or 돌파)이 다음 방향을 결정.

### 골든크로스/데드크로스 발생 시 국면 해석

| 신호 | 직후 국면 | 주의사항 |
|------|---------|---------|
| 골든크로스 발생 | 중장기 상승추세 전환 시작 | 이미 단기 랠리가 상당 부분 진행된 후 발생하는 경우 많음 (후행) |
| 골든크로스 이후 SMA200 위에서 가격 유지 | 상승추세 확인 | 건강한 강세장 |
| 골든크로스 이후 가격이 SMA200 아래로 재하락 | 골든크로스 실패 | 가짜 신호(False Break). 재평가 필요 |
| 데드크로스 발생 | 중장기 하락추세 전환 시작 | 단기 급락 이후 발생하는 경우 많음 (이미 많이 내린 뒤) |
| 데드크로스 이후 SMA200 아래에서 가격 유지 | 하락추세 확인 | 반등 시 SMA200을 저항선으로 활용 |

## 6. 시장 신호 도출 방법

### (1) 골든크로스 (Golden Cross) — 강세 신호

- **정의**: 단기 SMA가 장기 SMA를 위로 돌파하는 현상
- **가장 대표적인 형태**: 50일 SMA가 200일 SMA를 상향 돌파
- **의미**: 단기적 상승 모멘텀이 장기 추세보다 강해짐 → **강세(매수) 신호**
- **강화 조건**: 거래량이 높을 때 발생하면 신호 신뢰도 증가
- 시장 랠리(상승장)의 시작 가능성 시사

### (2) 데드크로스 (Death Cross) — 약세 신호

- **정의**: 단기 SMA가 장기 SMA를 아래로 돌파하는 현상
- **가장 대표적인 형태**: 50일 SMA가 200일 SMA를 하향 돌파
- **의미**: 단기적 하락 모멘텀이 장기 추세보다 강해짐 → **약세(매도) 신호**
- 기술적 분석에서 가장 주목받는 약세 패턴 중 하나

### (3) 가격과 SMA의 돌파 신호

- 가격이 SMA를 **위로 돌파** → 잠재적 강세 신호 (상승추세 시작 또는 재개)
- 가격이 SMA를 **아래로 돌파** → 잠재적 약세 신호 (하락추세 시작 또는 재개)

### (4) 복수 SMA 비교 신호

여러 기간의 SMA를 동시에 적용하여 신호의 신뢰도를 높인다.

| 비교 조합 | 일반적 활용 목적 |
|----------|----------------|
| 10일 / 50일 | 단기 추세 신호 |
| 20일 / 100일 | 중단기 추세 신호 |
| 50일 / 200일 | 장기 추세 신호 (골든크로스/데드크로스) |

## 7. 차트 분석 활용법

### 시각적 표현
- SMA는 **가격 차트에 직접 오버레이**되어 선 형태로 표시
- 기간이 길수록 선이 부드럽고 느리게 반응
- 복수의 SMA(예: 50일, 200일)를 함께 표시해 크로스오버 신호를 시각적으로 확인

### 차트에서 확인할 주요 패턴

| 패턴 | 가격 차트에서 보이는 모습 | 신호 |
|------|----------------------|------|
| SMA 우상향 | SMA 선이 왼쪽→오른쪽으로 상승하는 기울기 | 상승추세 확인 |
| SMA 우하향 | SMA 선이 왼쪽→오른쪽으로 하락하는 기울기 | 하락추세 확인 |
| 골든크로스 | 단기 SMA 선이 장기 SMA 선을 아래에서 위로 교차 | 강세 전환 |
| 데드크로스 | 단기 SMA 선이 장기 SMA 선을 위에서 아래로 교차 | 약세 전환 |
| 동적 지지선 반등 | 상승추세에서 가격이 SMA에 닿고 반등 | 매수 기회 |
| 동적 저항선 반락 | 하락추세에서 가격이 SMA까지 반등 후 다시 하락 | 매도 기회 |
| 가격이 SMA 위에 안착 | 가격선이 SMA 선 위에서 꾸준히 유지 | 상승추세 건강 확인 |
| 가격이 SMA 아래에 안착 | 가격선이 SMA 선 아래에서 꾸준히 유지 | 하락추세 건강 확인 |
| SMA 수렴 | 다수의 SMA 선이 한 점으로 모여드는 모습 | 횡보 또는 방향 결정 임박 |
| SMA 발산 | 다수의 SMA 선이 점점 벌어지는 모습 | 강한 추세 진행 중 |

### 차트 패턴 상세 설명

**SMA 지지/저항 테스트 패턴 (중요):**
- 가격이 SMA에 닿은 후 **캔들이 반등하면** 지지선으로 작동 확인 → 강세 유지
- 가격이 SMA에 닿은 후 **캔들이 아래로 마감하면** 지지선 붕괴 → 추가 하락 경고
- 하락추세에서 SMA에 닿았다가 **위로 돌파 실패하면** 저항선 확인 → 약세 유지

**SMA 밀집/수렴 패턴 (Convergence):**
- SMA20, SMA50, SMA200이 서로 가까워지면 → 방향성 부재 상태, 큰 움직임 예고
- 수렴 이후 어느 방향으로 SMA가 발산하느냐가 다음 추세의 방향

**SMA 배열 역전 패턴:**
- 상승추세 배열(price > SMA20 > SMA50 > SMA200)이 역배열로 바뀌는 과정:
  1. 먼저 SMA20이 SMA50을 하향 돌파 → 단기 경고
  2. 그다음 SMA50이 SMA200을 하향 돌파 (데드크로스) → 추세 전환 확인
- 이 순서를 차트에서 추적하면 추세 전환을 조기에 포착 가능

### 노이즈 필터링
- SMA는 원시 가격 차트보다 부드러운 선을 그려 일별 등락의 노이즈를 제거한다.
- 기간이 길수록 필터링 효과가 강해지며, 전반적인 추세 방향을 명확하게 파악할 수 있다.

## 8. SMA vs EMA (지수이동평균) 비교

| 항목 | SMA | EMA |
|------|-----|-----|
| 가중치 방식 | 모든 데이터에 동일 가중치 | 최근 데이터에 높은 가중치 |
| 반응 속도 | 느림 | 빠름 |
| 선의 특성 | 부드럽고 안정적 | 가격 움직임에 가깝게 따라옴 |
| 장점 | 잡음 적음, 장기 추세 파악에 용이 | 최근 변화를 빠르게 반영 |
| 단점 | 신호가 지연됨 | 잡음에 민감할 수 있음 |

## 9. 해석 시 주의사항 및 한계

### 후행 지표 (Lagging Indicator)
SMA는 과거 가격 데이터만을 기반으로 계산된다. 따라서:
- 추세 변화 신호가 항상 실제 가격 움직임보다 **늦게** 발생한다.
- 신호가 나올 때쯤에는 이미 움직임의 상당 부분이 진행된 경우가 많다.

### 균등 가중치의 한계
모든 날의 데이터를 동일하게 취급하므로, 오래된 데이터가 현재 시장 상황을 왜곡할 수 있다. 이로 인해 많은 트레이더가 EMA를 선호한다.

### 200일 SMA의 자기실현적 예언 위험
200일 SMA는 너무 많은 시장 참여자가 지켜보기 때문에, 이를 기반으로 한 집단 행동이 인위적인 가격 움직임을 만들어낼 수 있다. 이 지표에 지나치게 의존하는 것은 위험하다.

### 효율적 시장 가설(EMH)과의 충돌
SMA를 포함한 모든 기술적 분석은 과거 데이터를 기반으로 한다. 효율적 시장 가설에 따르면 현재 가격에 이미 모든 정보가 반영되어 있으므로, 과거 데이터 분석으로 초과 수익을 얻기 어렵다는 반론이 존재한다.
